# Fashion AI Project

## Overview
Fashion AI is a React-based application designed to provide users with various functionalities related to fashion, including image classification, recommendations, color analysis, and search capabilities. The application features a modern design with a deep black background and warm gold accents, creating an elegant user interface.

## Features
- **Hero Section**: An animated hero section that captures user attention.
- **Top Navigation Bar**: Easy navigation through different functionalities with tabs.
- **Two-Column Grid Layout**: Organized layout for input fields and results.
- **Image Uploads**: Users can upload images for analysis.
- **Flask Backend Integration**: Communicates with a Flask backend for processing requests.

## Setup Instructions
1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the frontend directory:
   ```
   cd fashion-ai-app/frontend
   ```
3. Install dependencies:
   ```
   npm install
   ```
4. Start the application:
   ```
   npm start
   ```

## Usage
- Navigate through the tabs in the top navigation bar to access different features.
- Upload images in the designated input fields to utilize the classification and recommendation functionalities.
- View results displayed in the organized grid layout.
